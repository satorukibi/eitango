# 英単語ドリル（ニュース頻出・中級）

スマホのブラウザで動く英単語学習アプリ（PWA）です。
AI・戦争・ニュースで頻出する中級レベルの英単語を学べます。

## 機能
- 1画面に **12個** の単語を表示
- 各単語：英単語 / 品詞 / カタカナ発音 / 例文 / 例文和訳 / 単語の意味
- 例文中の **対象単語を黄色でハイライト** 表示
- ヘッダーに **進捗バー**（覚えた割合 %）を表示
- 「🔊 読む」で単語・例文を音声で読み上げ（端末の音声合成を使用）
- 「**単語を全部読む**」「**例文を全部読む**」で12個を連続再生（読み上げ中のカードを強調表示）
- 「和訳を全部表示」「和訳を隠す」で一括操作
- 「別の12語」で進捗を変えずに別の未学習単語へ入れ替え
- 「英文を和訳する」「単語を和訳する」で意味を表示
- 「覚えた」を押すと、その単語が別の単語に入れ替わる
- 「覚えた一覧」から誤って覚えた単語を **戻す** ことができる
- 学習の進捗はスマホ内に自動保存（次回も続きから）
- オフライン対応（ホーム画面に追加でアプリのように使える）

## 使い方（スマホで使う）
音声読み上げや「ホーム画面に追加」を使うには、ローカルサーバー経由で開くのが確実です。

### PCで簡単に起動する
このフォルダで以下を実行（Pythonがある場合）：

```
python -m http.server 8000
```

スマホとPCが同じ Wi-Fi なら、スマホのブラウザで `http://<PCのIPアドレス>:8000/eitango/` を開きます。
（PCのIPは Windows のコマンド `ipconfig` で確認できます／例: 192.168.x.x）

### ホーム画面に追加
- iPhone(Safari)：共有ボタン →「ホーム画面に追加」
- Android(Chrome)：メニュー →「ホーム画面に追加」

## 単語を増やす・編集する
`words.js` の配列に追記するだけです。

```js
{ en: "単語", pos: "品詞", kana: "カタカナ", ja: "意味", ex: "例文", exJa: "例文の和訳" }
```

将来的に3000語規模へ拡張できます。
